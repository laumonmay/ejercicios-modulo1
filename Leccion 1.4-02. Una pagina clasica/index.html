Los 3 lenguajes estándar de programación

HTML: es un lenguaje de marcado de texto cuya misión es la de estructurar toda la información que se va a visualizar en un sitio web. Para ello, existen diferentes etiquetas que proporcionan al navegador datos relacionados con su semántica lo que permite clasificar al detalle cada apartado de una página web, por ejemplo se pueden detallar los encabezados, párrafos, artículos, imágenes, etc...

CSS: es un lenguaje que se utiliza para darle forma y color a la estructura del HTML. Con CSS aplicaremos estilos de letra, medidas, colores, adornos, etc...

JavaScript: es el lenguaje encargado de proporcionar interacción a la página web. Con el podremos enlazar nuestro sitio web con las bases de datos, recoger informacion de los usuarios y que estos participen de una forma dinámica.
